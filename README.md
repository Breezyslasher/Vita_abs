# VitaABS

A native Audiobookshelf client for PlayStation Vita enabling audiobook and podcast playback from your Audiobookshelf server with full controller support.

> Check the Issues page for current app feature status and known bugs.

## Features

**Library Management**
- Browse audiobook and podcast libraries with category tabs
- Item covers load from server
- Search across libraries
- Collections and series browsing

**Playback**
- Stream audiobooks and podcasts directly from server
- Play downloaded content offline
- Play/pause, seek forward/backward with configurable intervals
- Playback speed control (0.5x - 2.0x)
- Progress slider with elapsed/remaining time
- Cover art and metadata display in player

**Progress Sync**
- Automatic progress saving during playback
- Resume from last position
- Bidirectional sync with Audiobookshelf server
- Offline progress saves locally, syncs when reconnected

**Downloads**
- Download audiobooks and podcast episodes for offline playback
- Download queue with progress tracking
- Multi-file audiobook support
- Manage downloads from dedicated tab

**Podcast Support**
- Browse podcast libraries and episode lists
- Download individual or batch episodes
- Check for new episodes from server

**Settings**
- Theme selection (System/Light/Dark)
- Seek interval customization
- Download management options
- Debug logging toggle

## Requirements

- PlayStation Vita with HENkaku/Enso
- Running, accessible Audiobookshelf server
- Network connectivity between Vita and server

## Installation

1. Download the VPK file
2. Transfer via USB or FTP to Vita
3. Install using VitaShell
4. Configure Audiobookshelf server URL and login
